OpenStack Installation on Ubuntu

1. Prerequisites

You need a clean Ubuntu server installation with sufficient resources (RAM, CPU, and disk space) and internet access.

2. Update System

sudo apt update sudo apt upgrade -y

3. Install Git

sudo apt install git -y

4. Clone DevStack

git clone https://opendev.org/openstack/devstack.git

5. Navigate to DevStack Directory

cd devstack

6. Create a Local.conf File

Create a local.conf file in the devstack directory with your preferred configuration. Here's a minimal example:

touch local.conf cat <<EOF > local.conf
[[local|localrc]]
ADMIN_PASSWORD=secret
DATABASE_PASSWORD=\$ADMIN_PASSWORD
RABBIT_PASSWORD=\$ADMIN_PASSWORD
SERVICE_PASSWORD=\$ADMIN_PASSWORD
EOF

7. Run Stack.sh

./stack.sh

8. Wait for Installation

Follow the prompts and wait for the installation to complete.

9. Access OpenStack Dashboard (Horizon)

Once the installation is complete, access the OpenStack dashboard by opening a web browser and navigating to http://<your-server-ip>/dashboard.

10. Login

Use the username admin and the password you set in the local.conf file during installation.

11. Explore

You can now start using OpenStack to manage your cloud infrastructure. Explore the dashboard and the various features available.
