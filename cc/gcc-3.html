Step 1: Download Python

Download and install the latest version of Python for Windows from the official website:

Download Python

Step 2: Download Cloud SDK

Download and install Cloud SDK for Windows by following the quickstart guide:

Download Cloud SDK

During installation, it will ask for your email account for verification.

Step 3: Create YAML & Python File

3.1: Create app.yaml

Create a YAML file using Notepad, type the following content and save it as "app.yaml":


runtime: python27
api_version: 1
threadsafe: false
handlers:
- url: /
  script: index.py

3.2: Create index.py

Create a Python file named "index.py" with the following content:


print('Web service using Python code')
print('Information Technology - SOC - KARE')

Step 4: Open Google Cloud SDK Shell

Open "Google Cloud SDK Shell" from the Windows desktop.

Step 5: Run Development Server

Run the following command in the Google Cloud SDK Shell:


    cd C:\Users\sivak\AppData\Local\Google\Cloud SDK
    google-cloud-sdk\bin\dev_appserver.py "E:\KARE\2023-2024\Even Semester\cloud computing\cloud lab"

Note:

  1. C:\Users\sivak\AppData\Local\Google\Cloud SDK> This is the current working directory in the command prompt. It shows that you are currently in the "Cloud SDK" directory.
  2. google-cloud-sdk\bin\dev_appserver.py This is the relative path to the dev_appserver.py script within the Google Cloud SDK directory. This script is used to run the development server for Google App Engine applications.
  3. "E:\KARE\2023-2024\Even Semester\cloud computing\cloud lab" This is the absolute path to your Google App Engine application directory. The application you want to serve using the development server is located in this directory.

Step 6: Open URL in Browser

Open an internet browser and type the following URL:

http://localhost:8080/
