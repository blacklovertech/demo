Step 1: Install VirtualBox

To begin, download and install VirtualBox from its official website:

Direct Download VirtualBox

Step 2: Download Ubuntu ISO

Next, obtain the Ubuntu ISO file. You can download it from the official Ubuntu website:

Direct Download Ubuntu

USE Ubuntu Directly on the PC

Step 3: Create a New Virtual Machine

Once VirtualBox is installed, open it and create a new virtual machine:

  • Name: Ubuntu
  • Type: Linux
  • Version: Ubuntu (64-bit)
  • Allocate memory (RAM) and create a virtual hard disk

Step 4: Install Ubuntu

Now, you need to install Ubuntu on your virtual machine:

  1. Launch VirtualBox and select the Ubuntu virtual machine.
  2. Click on the green "Start" arrow to launch the virtual machine.
  3. When prompted, select the Ubuntu ISO file you downloaded as the bootable media.
  4. Follow the on-screen instructions to install Ubuntu.

Step 5: Open Demo

After Ubuntu is installed, you can open a demo HTML file:

  1. Launch Ubuntu in VirtualBox.
  2. Open a web browser within Ubuntu.
  3. Navigate to the location where your HTML file is stored.
  4. Open the HTML file in the web browser to view its content.
