Transfer File Between Two VirtualBox Ubuntu Machines

Step 1: Create VM-1 and VM-2

Create two VirtualBox Ubuntu VMs named "VM-1" and "VM-2" with suitable configurations.

Step 2: Start VM-1 and VM-2

Start both VM-1 and VM-2 in VirtualBox.

Step 3: Configure Networking

Ensure both VMs are using the same network mode in VirtualBox settings. You can choose either "NAT Network" or "Bridged Adapter".

Step 4: Obtain IP Addresses

In VM-1, open a terminal and run:

ip addr show

Note down the IP address of VM-1. Repeat the same process in VM-2 to obtain its IP address.

Step 5: Transfer File from VM-1 to VM-2

Assuming you have a file named "example.txt" on VM-1 that you want to transfer to VM-2:

scp /path/to/example.txt username@VM2_IP:/path/to/destination

Replace /path/to/example.txt with the path to the file on VM-1, username with your username on VM-2, VM2_IP with the IP address of VM-2, and /path/to/destination with the desired destination path on VM-2.

After executing the command, you will be prompted to enter the password for your username on VM-2.

Step 6: Verify File Transfer

Once the transfer is complete, log in to VM-2 and navigate to the destination directory to verify that the file has been successfully transferred.

With these steps, you should have successfully transferred a file from VM-1 to VM-2.
