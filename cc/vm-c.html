Step 1: Install VirtualBox

Download and install VirtualBox from the official website:

Download VirtualBox

Step 2: Download Ubuntu ISO

Obtain the Ubuntu ISO file from the official Ubuntu website:

Download Ubuntu ISO

Step 3: Create a New Virtual Machine

Open VirtualBox and create a new virtual machine:

  • Name: Ubuntu
  • Type: Linux
  • Version: Ubuntu (64-bit)
  • Allocate memory (RAM) and create a virtual hard disk

Step 4: Install Ubuntu

Boot the virtual machine with the Ubuntu ISO file and install Ubuntu:

  1. Launch VirtualBox and select the Ubuntu virtual machine.
  2. Click on the green "Start" arrow to launch the virtual machine.
  3. When prompted, select the Ubuntu ISO file you downloaded as the bootable media.
  4. Follow the on-screen instructions to install Ubuntu.

Step 5: Write and Compile Your C Program

Once Ubuntu is installed and running in the virtual machine, follow these steps:

  1. Open a terminal in Ubuntu.
  2. Write your C program using a text editor (e.g., nano, vim). For example, create a file named hello.c:
    3. #include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}
  3. Save the file.
  4. Compile your C program using the gcc compiler. For example: 
    gcc hello.c -o hello

Step 6: Run Your C Program

After successfully compiling your C program, execute it in the terminal:

  1. Navigate to the directory containing the executable.
  2. Run the executable by typing its name and pressing Enter. For example: 
    ./hello
  3. You should see the output Hello, world! printed in the terminal.
