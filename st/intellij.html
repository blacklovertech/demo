1. Launch IntelliJ IDEA:
    • Open IntelliJ IDEA IDE on your system.
  2. Create a New Project:
    • Click on "File" in the menu.
    • Choose "New" and then "Project".
    • Select the type of project you want to create (for example, Java).
    • Click "Next".
    • Enter a name for your project.
    • Choose the project SDK (Java version).
    • Click "Finish".
  3. Add Selenium's .jar files as External Libraries:
    • Download the Selenium WebDriver Java bindings from the Selenium website or add the Selenium dependency to your Maven or Gradle project.
    • Once downloaded, extract the contents if necessary.
    • In IntelliJ IDEA, right-click on your project in the "Project" view.
    • Choose "Open Module Settings" or press "F4".
    • Click on "Libraries" under "Project Settings".
    • Click on the "+" button and select "Java".
    • Navigate to the location where you saved the Selenium WebDriver Java bindings.
    • Select the .jar files and click "OK" to add them as external libraries.
  4. Configure Maven as a Dependency Manager (Optional):
    • If you want to use Maven to manage dependencies (including Selenium), you can configure it in IntelliJ IDEA.
    • Open the "Project Structure" window again (File -> Project Structure or Ctrl+Alt+Shift+S).
    • Click on "Project" under "Project Settings".
    • Set the "Project SDK" and "Project language level" if not already set.
    • Click on "Modules" under "Project Settings".
    • Select your module and click on the "Dependencies" tab.
    • Click on the "+" button and select "1. From Maven".
    • Search for the Selenium dependency you want to add and click "OK" to add it to your project.

After completing these steps, you should be ready to start writing Selenium tests in IntelliJ IDEA. You can create new Java classes and start writing your Selenium WebDriver code to automate browser interactions.



Initialize the Browser

1.1 Initialize the Browser (Chrome/Firefox/Edge/IE11):

// Initializing the browser driver
    WebDriver driver = new ChromeDriver();

1.2 Get the URL:

// Navigating through a particular website
    driver.get("https://www.codoid.com");

Browser Navigation

2.1 Navigate to:

driver.navigate().to("https://www.QA-Codoidian.com");

2.2 Back, Forward, Refresh command:

// Pressing the browser’s back button:
    driver.navigate().back();
    
    // Pressing the browser’s forward button:
    driver.navigate().forward();
    
    // Refresh the current page:
    driver.navigate().refresh();

Get Browser Information

3.1 Get title:

// Read the current page title from the browser
    driver.getTitle();

3.2 Get Current URL:

// Retrieve the current URL from the browser’s address bar
    driver.getCurrentUrl();

3.3 Get Page Source:

// Get the current page source and it’ll return the string type
    String strPageSource = driver.getPageSource();

FindElement vs FindElements

i. Why do you need the Find Element command?

// Interaction with a web page requires a user to locate the web element, and this command is used to uniquely identify a single web element in the current web page.
    Example: driver.findElement(By.id("username"));

ii. FindElements command:

// FindElements command is used to uniquely identify the list of web elements within the web page.
    Example: driver.findElements(By.id("name"));

Frames

public static void main(String[] args) throws InterruptedException {
    System.out.println("-------> KLU");
    WebDriver driver = new ChromeDriver();
    driver.get("https://jqueryui.com/droppable/");
    Thread.sleep(3000);
    driver.manage().window().maximize();
    WebElement frame = driver.findElement(By.xpath("//iframe[@class='demo-frame']"));
    driver.switchTo().frame(frame);
    WebElement DragBox = driver.findElement(By.id("draggable"));
    WebElement drop = driver.findElement(By.id("droppable"));
    Actions act = new Actions(driver);
    act.dragAndDrop(DragBox, drop);
    act.build().perform();
    driver.switchTo().defaultContent();
    WebElement SortLink = driver.findElement(By.xpath("//a[text()='Sortable']"));
    SortLink.click();
    }
